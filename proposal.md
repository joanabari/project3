# PROPOSAL.md

## What I'm building
I am building a Pokédex web app that lets users browse, search, and save their favorite Pokémon.

## Which API I'm using
PokéAPI  
https://pokeapi.co/

## Why I chose this
I chose this API because I like Pokémon, and the data is interesting, visual, and easy to explore in a web app.

## Core features
- Browse a list of Pokémon
- Search for a Pokémon by name
- Display Pokémon cards with image, name, and type
- Show loading and error messages while fetching data
- Save favorite Pokémon using localStorage

## What I don't know yet
I still want more practice with fetch(), async/await, handling JSON data, and connecting user input to API calls.
